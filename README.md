# 🏥 AarogyamScanAI: Intelligent Medical Imaging Diagnostics

**AarogyamScanAI** ek AI-based medical analysis tool hai jo X-rays, MRI, aur CT scans ko scan karke instant reports generate karta hai. Iska maqsad healthcare mein fast aur accurate diagnosis provide karna hai.

## 🚀 Key Features
* **Multi-Modal Support:** X-ray, MRI, aur CT scan ka analysis.
* **Disease Detection:**
  * **X-ray:** Pneumonia, Tuberculosis, aur Tumors ki pehchan.
  * **MRI:** Brain Tumor detection.
  * **CT Scan:** Kidney Stone analysis.
* **Automated Reporting:** Scan ke baad PDF/Image format mein analysis report.
* **User History:** Pichle saare scans aur results ka record.

## 🛠️ Tech Stack
* **Backend:** Flask (Python)
* **Frontend:** HTML5, CSS3, JavaScript
* **AI/ML:** TensorFlow/Keras (CNN Models)
* **Database:** SQLite

## 📁 Project Structure

- `models/`: Trained AI models (.h5 files) aur training scripts.
- `static/`: CSS, JS, aur images.
- `templates/`: HTML pages (Index, MRI, CT Scan, etc.)
- `utils/`: Prediction logic aur helper functions.
